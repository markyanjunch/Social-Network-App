# social-network-app
A social network website for Developers built with MERN Stack  
## Demo 
https://frozen-chamber-77332.herokuapp.com/  
## Supports
- user login/register
- CRUD operations on user profiles
- like/unlike/comment on posts
- recent GitHub repos display  
- ...
